# 3.1 Compute Types

Azure offers wide variety of compute types depending on the infrastructural needs.

Compute types are **Flexible and on demand.**

These include:

- Virtual Machines
- App Services
- Container Instances
- Azure Kubernetes Services
- Azure Virtual Desktop
- Function Apps
